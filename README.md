# Business Intelligence Dashboard

This project is a Business Intelligence Dashboard built using Streamlit, Pandas, and Plotly.

The dashboard analyzes company data including revenue, expenses, profit, and employee productivity.

Features:
- KPI metrics
- Revenue trend analysis
- Region performance
- Department performance
- Employee productivity
- Interactive filters

Technologies Used:
Python
Streamlit
Pandas
Plotly

Run locally:

streamlit run app.py
